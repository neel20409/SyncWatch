# SyncWatch
Watch YouTube videos in real-time sync with friends — built with Next.js, Socket.io &amp; MongoDB
A full-stack watch party app where two or more people can watch any YouTube video together in perfect sync from different locations. One person controls playback — play, pause, seek — and it instantly syncs for everyone in the room.

Built with Next.js, Socket.io for real-time communication, and MongoDB for user authentication. Features include room creation with shareable codes, live chat, and JWT-based login/signup.
